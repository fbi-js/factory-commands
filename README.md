# factory-commands

Global commands for fbi

## use

add fbi global

```shell
$ npm i -g fbi@next
# OR
$ yarn global add fbi@next
# OR
$ pnpm i -g fbi@next
```

add factory-commands

```shell
$ fbi add factory-commands

$ fbi c
# OR
$ fbi commit
```

then you can use all commands in this factory anywhere
